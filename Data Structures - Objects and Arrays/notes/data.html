Intro to Data Structures

What is a Data Structure?

When an application needs to keep track of data that is more complex than what can be stored in primitive types, we create objects which can store more complicated data structures.

Objects

Objects are like containers. Inside the containers are properties, which are referred to as key-value pairs. 

      
        Example object:
        const myObject = {
          propertyOneName: PropertyOneValue,
          propertyTwoName: PropertyTwoValue,
        }
        // Inside the curly braces is called the object literal

Arrays

Arrays are a type of object which stores data when the order of the data matters.
